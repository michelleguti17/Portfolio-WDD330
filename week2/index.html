Scope

Using const and let to declare variables means they are block scoped, so their value only exists inside the block they are declared in.There are two types of scopes Global and Local. Global variables are declared outside of a block and they are accesible in anywhere in the program. Local variables are defined inside a block and will only be available inside that block and not be accessible outside of that block.The words let or const should be used to delcare a local variable.

Type Coercion

Type coercion happens when the operands of an operator are of different data types. JavaScript try to convert one operand type to an equivalent value of the other operand's type.

Converting string and numbers

There are way to convert strings to number and viceversa with the numbers(), string(), and tostring() functions.The parseInt() also convert a string to a number, it is common to use when there is an input from the user.

Template strings

Template literals use the backtick character, `, to deliminate the string. They allow interpolation which means that a JS expression can be inserted inside a string.

What is an object?

An object is a definition in JS that is populated with different properties or keys that are the same thing. In the case of a human, a person is an object, and its properties are its age, name, skin, height, etc. The properties of an object are like the characteristics of a human, a thing, or an animal.

Arrays

Arrays are used to store lists. The objects inside and the lenght of an array are dynamic so they can also change in runtime. To declare and array we use [] it is important to remember that in JS we can combine different data types in the same array

To create arrays literal we just need to create an array with initial values separated by a comma inside the square braquets.

const colors = ['blue', 'yellow,' 'red'];

Arrays properties and methods

Arrays has different properties or concepts which help with arrays modifications.

      delete: Remove an item from an array.
      lenght: To find the length of an array.
      pop(): To remove the last item from an array.
      shift(): To remove the first item in the array.
      push(): To append a new value to the end of the array.
      unshift(): To append a new item to the beginning of the array.
      concat():To merge an array with one or more arrays.
      spread: The spread operator is three dots, ... that are placed in front of an array, with the effect of spreading out the elements of two arrays and arrange them in a new array.
      join(): To turn the array into a string that comprises all the items in the array, separated by commas. We can also choose a separator other than comma by placing it inside the parentheses.
      slice(): It creates a sub array, by taking a slice of the original array, starting at one position and finishing at another.
      splice(): To remove items from an array then insert new items in their place. Also to insert values into an array at a specific index without removing any items.
      reverse(): To reverse the order of the array.
      sort(): To sort the order of an array.
      indexOf(): To find the first occurrence of a value in an array.
      includes(): To return a boolean value depending on whether the array contains a particular element or not.

Sets

A set is a data structure that represents a collection of unique values, so it cannot include any duplicate values.

To create a set we use new operator and Set() with parantheses. To add values to a set we use the name of the set with the add()method or by placing the items inside an array

const numberList =new Set();

Add Method

numberList.add(1).add(2).add(3);

Inside an Array

const numberList = new Set([1,2,3]);

Set Methods
      size(): The number of values found in a set.
      has(): To check if a value is in a set. Sets do not have index notation for inspecting individual entries
      delete(): To remove a value from a set.
      clear():To remove all values from a set.
Maps

The keep a list of key and value pairs. Maps can use any data type as a key.They can only storage and retrieve key-value pairs.

To create a map we use new operator and Map() constructor. To add values to a map we use the name of the map with the set()method.The set() method can be used to add a key and value pair to a map. The first value is the key and the second is the value

const vowels =new Maps();

Set Method inside Maps

vowels.set(1, 'a').set(2, 'e').set(3, 'i').set(4, 'o').set(5, 'u');

To cresate a map with multiple values we can use a nested array as a parameter.

const vowels = new Map([ ['a','e'],['i',o','u']]);

Maps Methods
      get(): To retrieve any values
      has(): To check if a particular key is in a map.This returns a boolean value of true or false
      size: To find the number of key and value pairs in a map.
      delete(): To remove a key and value pair from a map.This returns a boolean value of true if the value was removed or false if it wasn't in the map.
      clear(): To remove all key and value pairs from a map

Callbacks

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

The callback is provided as a parameter, then invoked inside the body of the function.

function hello(name,callback){
console.log(`Lello ${name}.`);
callback();
}
